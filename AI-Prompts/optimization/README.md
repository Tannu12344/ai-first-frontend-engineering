# Performance Optimization Prompts

## Purpose

Analyze and optimize React applications for better performance and scalability.

---

# Example Prompt

```txt
Optimize this React application for performance.

Analyze:
- unnecessary re-renders
- state management issues
- memoization opportunities
- bundle size
- lazy loading
- API optimization
- virtualization opportunities

Code:
[PASTE CODE]
```

---

# Focus Areas

* React.memo
* useMemo
* useCallback
* lazy loading
* code splitting
* rendering optimization
* API efficiency
