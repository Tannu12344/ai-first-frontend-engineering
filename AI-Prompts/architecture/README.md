# Frontend Architecture Prompts

## Purpose

Generate scalable frontend folder structures and architecture systems for React and Next.js applications.

---

# Example Prompt

```txt
Design scalable frontend architecture for [PROJECT NAME].

Requirements:
- feature-based architecture
- reusable components
- centralized API layer
- authentication flow
- routing architecture
- scalable folder structure
- performance optimization
- deployment considerations

Include:
1. Folder structure
2. State management strategy
3. API architecture
4. Reusability strategy
5. Scaling strategy
```

---

# Best Use Cases

* SaaS applications
* Dashboards
* Enterprise frontend systems
* Large React applications
* Next.js projects
