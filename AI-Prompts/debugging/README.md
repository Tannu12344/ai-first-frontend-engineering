# React Debugging Prompts

## Purpose

Identify root causes of React issues and generate scalable fixes.

---

# Example Prompt

```txt
Analyze this React issue deeply.

Problem:
[PASTE ISSUE]

Code:
[PASTE CODE]

Tasks:
- Find root cause
- Explain why it happens
- Fix the issue
- Improve code quality
- Suggest optimizations
- Mention best practices violated
```

---

# Best Use Cases

* Rendering bugs
* State issues
* API failures
* Hook dependency issues
* Performance bugs
* Form validation bugs
