# Component Generation Prompts

## Purpose

Generate scalable and reusable React components faster using AI-assisted workflows.

---

# Example Prompt

```txt
Create a reusable React component for [FEATURE].

Requirements:
- React + TypeScript
- Responsive design
- Tailwind CSS or Material UI
- Proper loading states
- Error handling
- Accessibility support
- Reusable props
- Clean architecture
- Production-ready code

Also include:
- folder structure
- interfaces/types
- custom hooks if needed
- optimization suggestions
```

---

# Best Use Cases

* Forms
* Modals
* Cards
* Tables
* Dashboards
* Navigation bars
* Charts
* Reusable UI systems
